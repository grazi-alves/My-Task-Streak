My Task Streak é um rastreador de tarefas desenvolvido com **JavaScript puro**, focado em organização, consistência diária e boa estrutura de código.

## Funcionalidades
- Criação e gerenciamento de tarefas
- Listas separadas (Tarefas Diárias, Projetos e Faculdade)
- Contadores em tempo real (total, feitas e pendentes)
- Interface simples e responsiva

## Tecnologias
- HTML5
- CSS3
- JavaScript (Vanilla JS)

## Objetivo
Projeto desenvolvido para praticar fundamentos de frontend, manipulação do DOM e organização de estados da interface, servindo como base para evoluções futuras.
